# Privacy

ElyraLauncher is designed to work locally by default.

## Data Collection

- No telemetry.
- No hidden analytics.
- No hidden network calls.
- No proprietary cloud dependency.

## Search and Suggestions

Launcher search and app suggestions are local-first. Optional external providers
must be disabled by default, clearly labeled, and documented before release.

## Local Data

Launcher settings, layout state, hidden app choices, and layout snapshots are
stored locally on the device using Android storage appropriate for launcher
state.

## Crash Handling

Crash diagnostics may be written locally for debugging. They are not uploaded
automatically.
